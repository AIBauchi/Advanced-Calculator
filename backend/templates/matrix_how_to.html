How to pass matrices

Pass matrix rows as lists in a list

Pass two matrices as separated by a comma(,)

Sample Input:

Passing one matrix

2 by 2 matrix:

[[1, 2], [3, 4]]

3 by 3 matrix:

[[1, 2, 4], [3, 4, 7], [4, 1, 7]]

Passing two matrices

2 by 2 matrix:

[[1, 2], [3, 4]], [[4, 1], [3, 2]]

3 by 3 matrix:

[[1, 2, 4], [3, 4, 7], [4, 1, 7]], [[1, 2, 4], [3, 4, 7], [4, 1, 7]]
